# manjunatha-stores

 Manjunatha Stores Online
A lightweight, high-performance grocery web app built for Manjunatha Stores. This project was developed using a collaborative Human-AI workflow, leveraging AI for rapid prototyping, logic generation, and automated marketing assets.



✨ Features
Live Inventory: Real-time product syncing from Google Sheets.
One-Tap Ordering: Seamless WhatsApp integration for order fulfillment.
Zero Login: No account creation required—just name and address.
Local Persistence: Remembers user delivery details for future orders.
Category Filtering: Instant browsing for Eggs, Spices, Dairy, Fruits, and more.
Smart Cart: Dynamic delivery fee calculations based on order value.



🛠️ Built With
Frontend: HTML5, CSS3 (Poppins Typography), JavaScript (ES6+).
Backend (Data): Google Sheets API (CSV Export logic).
Analytics: Google Tag Manager (GTag).
AI Collaborators: * Logic & Code: Gemini (Google AI).
Marketing & Voiceover: ElevenLabs (Anvi Voice).
Creative Assets: Midjourney/DALL-E for background generation.



🚀 AI-Enhanced Workflow
This project is a case study in using AI to bridge the gap between a local physical store and a digital presence:
Code Generation: AI assisted in writing the Google Sheet parsing logic and dynamic DOM rendering.
Asset Creation: Video ads were created using AI-generated background environments and AI voice clones to mimic professional high-budget commercials.
Optimization: AI was used to debug cross-platform mobile responsiveness and logic for the WhatsApp message formatting.
